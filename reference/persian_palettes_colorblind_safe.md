# Colorblind-Safe Persian Color Palettes

A subset of Persian color palettes that are colorblind-safe. These
palettes have been tested for accessibility, see
[`vignette("check-colors")`](https://jan-yegi.github.io/peRsian/articles/check-colors.md).

## Usage

``` r
persian_palettes_colorblind_safe
```

## Format

A named list of character vectors containing hex color codes
